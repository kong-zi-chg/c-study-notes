## 2. 学习主线（基础课如何接上 AI 时代）

### 2.1 传统课程不是过时，而是 AI 时代的“地基”

- **高数**：优化、梯度下降、损失函数理解全靠它  
- **线代（后续要补）**：向量、矩阵是深度学习语言  
- **概率统计（后续要补）**：模型评估、不确定性、贝叶斯思维  
- **数据结构与算法**：决定代码效率与思维上限  
- **C/Python**：C 训练底层思维，Python 快速落地 AI 应用

一句话：**AI 是高楼，数理和算法是地基，工程能力是钢筋。**

### 2.2 你的课程与 AI 对接图

1. C 语言数据结构  
   -> Python 重写同一数据结构  
   -> 对比时间复杂度与可读性  
   -> 用在小项目中（如搜索、推荐、路径规划）

2. Python 基础  
   -> 数据处理（pandas）  
   -> 可视化（matplotlib）  
   -> 机器学习入门（scikit-learn）  
   -> 大模型应用（API 调用 + Prompt 工程 + RAG 入门）

3. 高数补弱  
   -> 每学一个概念就做一个“算法关联笔记”  
   例如：导数 -> 梯度下降为什么能找最优值

## 8. AI 时代学习原则（防止被工具“反向驯化”）

1. **先思考再问 AI**：先写自己的答案，再用 AI 对照
2. **让 AI 当教练，不当代写**：让它提示思路，不直接抄代码
3. **所有 AI 输出都要验证**：能跑、能解释、能改错才算会
4. **建立你的提示词库**：记录“什么问法最有效”
5. **每周一个 AI+课程融合点**：例如“用 AI 解释红黑树并手写实现”

## 15. 路径决策表（考研权重 vs 实习权重）

> 目标：不靠情绪做决定，用打分决定“当前阶段时间怎么分配”。

### 15.1 评分维度（每项 0-10 分）

1. 数学基础（高数/线代/概率）  
2. 数据结构与算法基础  
3. 编程实现能力（C/Python）  
4. 项目作品成熟度（可展示、可讲解）  
5. 英语能力（阅读技术资料、考试基础）  
6. 学校平台与资源（导师/实验室/比赛机会）  
7. 短期实习可得性（是否容易拿到实习机会）  
8. 抗压与执行稳定性（能否长期坚持）

### 15.2 决策公式（简单版）

- `考研准备指数 G = (1+2+5+8)/4`
- `实习冲刺指数 I = (2+3+4+7+8)/5`

判定规则：
- 若 `G - I >= 1.5`：当前阶段以考研主线为主（建议 70/30）
- 若 `I - G >= 1.5`：当前阶段以实习主线为主（建议 40/60）
- 若 `|G - I| < 1.5`：双线并行（建议 60/40）

> 注：这里的“70/30”指学习时间分配，不是放弃另一条路。

### 15.3 你的当前默认评分（2026-03-11 初版）

基于你目前反馈（大一下、数学薄弱、基础在补、目标大厂+升学）：

- 数学基础：`3`
- 数据结构与算法基础：`4`
- 编程实现能力：`5`
- 项目成熟度：`2`
- 英语能力：`5`（暂估）
- 学校资源利用：`4`（暂估）
- 实习可得性：`3`
- 执行稳定性：`6`

计算：
- `G = (3+4+5+6)/4 = 4.5`
- `I = (4+5+2+3+6)/5 = 4.0`

结论（当前阶段）：
- `G - I = 0.5`，属于 `|G-I| < 1.5`
- **建议：双线并行，时间配比 60%（考研基础） / 40%（工程与实习能力）**

## 19. 今日绕点总复盘（2026-03-11，C/数据结构）

> 用法：每天先看这页 3 分钟，再开始写代码和做题。

### 19.1 概念最容易混的点（一句话版）

1. **复杂度**：先找 `n`，再看循环变量怎么变（`i++`/`i*=2`/`i*i<=n`），嵌套就相乘。  
2. **`a`、`p`、`*p`、`&p`**：`p`是地址值，`*p`是该地址处的值，`&p`是指针变量自己的地址。  
3. **`a` 与 `&a`**：地址数字常看起来一样，但类型不同，`a+1` 和 `&a+1` 步长不同。  
4. **`%p` 与 `&`**：`&`负责取地址，`%p`只负责“打印地址”。  
5. **`void*`**：通用地址容器，不能直接解引用，必须先强转为具体类型。  
6. **`qsort`**：传进 compare 的是“元素地址”，不是元素值。  
7. **字符与字节**：1字节永远8位；字符占几字节由编码决定。  
8. **中文字符**：中文是字符，但在UTF-8里常是多字节，不适合单个 `char` 直接存。  
9. **`\r` vs `\n`**：`\r`回到行首会覆盖，`\n`才是换行。  
10. **`sizeof` vs `strlen`**：`sizeof`看容量字节数；`strlen`看实际字符串长度（不含`\0`）。  
11. **数组与结构体字段**：`int a[20]` 和 `char name[20]` 都是20格数组，只是每格类型不同。  
12. **`strcpy` 使用范围**：只对 `char[]` 字符串有效，不能拷贝到 `int[]/float[]`。  
13. **链接错误 `ld`**：本地可执行程序需要 `main`，否则链接失败。
