# C 学习笔记（持续更新中）

这是我个人的 C 语言、数据结构和基础 Python 学习仓库。  
目标很明确：把基础打扎实，记录真实进步，后续用于升学和技术能力展示。

## 这个仓库在做什么

- 记录每天/每周学了什么，不只放答案
- 保留踩坑和修复过程，方便复盘
- 放可运行的小代码，慢慢积累成体系

## 我现在的学习方式

- 先自己思考和动手写
- 再用 AI 一起做讲解、纠错和复盘
- 重点不是“抄出来”，而是“看懂并能改”

## 说明

这个仓库还在持续整理中，内容会逐步规范。  
如果你也在学基础，希望这份记录能给你一点参考。
