## 5. 第1课：极限（超白话版本）
### 5.1 极限到底是什么
极限就是：**x越来越靠近某个值时，f(x)会越来越靠近哪个数。**

例子：  
当 x -> 2 时，f(x)=x^2 的值会靠近 4。  
写作：lim(x->2) x^2 = 4

现在必须记住的话：**极限关心的是“靠近趋势”，不是“必须取到”。**

### 5.2 你最容易卡的点
- 卡点1：把“x->a”理解成“x=a”。
- 卡点2：看到分母为0就直接慌。
- 卡点3：不会判断该代入、化简还是用公式。

现在必须记住的话：**能直接代入就代入；不能代入再化简。**

### 5.3 三种常见起手法
1. 直接代入法：代入后不是 0/0、∞/∞ 就结束。  
2. 因式分解法：出现 0/0 时，先看能不能约掉。  
3. 有理化法：根号型 0/0，常用“乘共轭”。

现在必须记住的话：**0/0 不是答案，它只是提醒你“还没化简完”。**

### 5.4 极限最小题组（先做这3题）
1) lim(x->3) (2x+1)  
2) lim(x->2) (x^2-4)/(x-2)  
3) lim(x->0) (sqrt(1+x)-1)/x

参考答案（先自己做再看）：
- 1) 7  
- 2) 4（先分解：(x-2)(x+2)/(x-2)=x+2）  
- 3) 1/2（乘共轭后化简）

## 9. 第2课：积分入门（你现在就需要的版本）
### 9.1 一句话理解“积分”
积分就是：**把很多很小的量加起来，得到总量。**

你可以把它理解成“求和的连续版本”。

现在必须记住的话：**导数是“看变化快慢”，积分是“把变化累计起来”。**

### 9.2 先学不定积分（最常考、最先上手）
不定积分写作：
∫f(x)dx = F(x) + C

含义是：找一个函数 F(x)，让它的导数等于 f(x)。

也就是：
如果 F'(x)=f(x)，那么 ∫f(x)dx=F(x)+C

这里的 C 是任意常数，不能丢。

现在必须记住的话：**不定积分本质上是“导数的逆运算”。**

### 9.3 你最容易犯的3个错误
1. 忘记 +C。  
2. 把 ∫x^n dx 写错成 x^n/(n+1)（少了幂次加1）。  
3. 把积分和导数公式混着用，符号一紧张就乱。

现在必须记住的话：**每做完一道不定积分，反向求导检查一次。**

### 9.4 必背基础公式（第一批）
1. ∫x^n dx = x^(n+1)/(n+1) + C（n≠-1）  
2. ∫(1/x)dx = ln|x| + C  
3. ∫e^x dx = e^x + C  
4. ∫sin x dx = -cos x + C  
5. ∫cos x dx = sin x + C

现在必须记住的话：**积分提速的第一步就是“先背最小公式库”。**

### 9.5 立刻会做的3道题（不定积分）
1) ∫(3x^2+2)dx  
2) ∫(1/x)dx  
3) ∫(2e^x-cos x)dx

参考答案（先自己做）：
- 1) x^3+2x+C  
- 2) ln|x|+C  
- 3) 2e^x-sin x+C

---

### 9.6 再认识定积分（直觉先行）
定积分写作：
∫[a,b] f(x)dx

它表示在区间 [a,b] 上的“累计量”。  
当 f(x)≥0 时，可理解为函数图像与x轴围成的面积（几何意义）。

现在必须记住的话：**定积分有上下限，不定积分没有上下限。**

### 9.7 微积分基本定理（先会用，不死抠证明）
如果 F'(x)=f(x)，那么：
∫[a,b] f(x)dx = F(b)-F(a)

这句话非常关键：  
**定积分可以通过原函数代上下限来算。**

现在必须记住的话：**先求原函数，再“上限减下限”。**

### 9.8 定积分最小题组
1) ∫[0,1] 2x dx  
2) ∫[1,2] (3x^2) dx  
3) ∫[0,π] sin x dx

参考答案：
- 1) 1  
- 2) 7  
- 3) 2

## 10. 积分做题流程（专治“看会做不会”）
1. 先判断：这是不定积分还是定积分？  
2. 不定积分：先匹配公式，再保留 +C。  
3. 定积分：先求原函数 F(x)，再代上限减下限。  
4. 最后检查：符号是否错、常数是否漏、上下限是否代反。  

现在必须记住的话：**积分题先分类，90%的混乱会消失。**
